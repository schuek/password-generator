# Password Generator  
Generador de contraseñas personalizables con JavaScript.

##  Funcionalidades
- Elegir longitud de la contraseña.
- Activar/desactivar:
  - Mayúsculas  
  - Minúsculas  
  - Números  
  - Símbolos
- Copiar contraseña al portapapeles.

##  Tecnologías
pyhton

##  Aprendizaje
- Manejo de funciones según preferencias del usuario.
- Generación aleatoria de caracteres.
- Validaciones básicas.
- Buenas prácticas en proyectos pequeños.

##  Ejecución
Abrir `app.py` en el navegador.
